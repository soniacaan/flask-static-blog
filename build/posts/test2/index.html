Flask is a microframework for Python based on Werkzeug, Jinja 2 and good intentions. And before you ask: It's BSD licensed! Latest Version: 0.12.2 Flask is Fun from flask import Flask app = Flask(name)

@app.route("/") def hello(): return "Hello World!" And Easy to Setup $ pip install Flask $ FLASK_APP=hello.py flask run * Running on http://localhost:5000/
